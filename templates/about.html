Про проєкт

Система пошуку найкоротших шляхів

Курсова робота з курсу "Бази даних та сховища даних" Львівського національного університету імені Івана Франка.

Проєкт демонструє застосування ройових алгоритмів (Ant Colony Optimization) для розв'язання класичної задачі пошуку найкоротшого шляху в графах з використанням NoSQL бази даних MongoDB.

Використані технології
Backend:
  • Python 3.10+
  • Flask (веб-фреймворк)
  • PyMongo (драйвер MongoDB)
  • NumPy (обчислення)
Frontend:
  • Bootstrap 5
  • Vis.js (візуалізація)
  • Chart.js (графіки)
  • jQuery
База даних:
  • MongoDB (NoSQL)
Реалізовані алгоритми
Ant Colony Optimization (ACO)

Мурашиний алгоритм - метаевристичний алгоритм оптимізації, що імітує поведінку мурах під час пошуку шляху до їжі. Основна ідея - використання феромонів для комунікації між агентами.

Алгоритм Дейкстри

Класичний алгоритм пошуку найкоротшого шляху в графі з невід'ємними вагами ребер. Використовується для порівняння ефективності з ACO.

Функціонал системи
  • Інтерактивне створення та редагування графів
  • Генерація випадкових графів
  • Генерація графа міст України
  • Пошук найкоротшого шляху з використанням ACO
  • Пошук за алгоритмом Дейкстри
  • Порівняння ефективності алгоритмів
  • Візуалізація графів та знайдених шляхів
  • Налаштування параметрів алгоритмів
  • Збереження історії пошуків
  • Статистика використання системи
